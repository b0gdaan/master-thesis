# LaTeX teza

Ova fascikla sadrzi pocetni kostur master rada zasnovan na klasi `finthesis` preporucenoj od strane profesora.

Sta je vec uradjeno:
- preuzet je `finthesis.cls` i prateci assets
- napravljen je kostur rada sa poglavljima
- unet je pocetni tekst zasnovan na stvarnim rezultatima iz `outputs/`
- dodata je pocetna bibliografija

Sta obavezno treba dopuniti pre konacne predaje:
- broj indeksa
- modul
- mentor i puni naziv mentora
- clanovi komisije
- prijava rada ako bude trazena
- dorada bibliografije na najmanje 10 jakih jedinica, od kojih bar 5 novijih

Preporuceno kompajliranje:
- XeLaTeX ili PDFLaTeX + Biber, u zavisnosti od lokalnog okruzenja
- Overleaf je takodje moguca opcija
